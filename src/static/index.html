Welcome to the Googleart2RDF server.

The GoogleArtProject is a place to explore museums from around the world, seeing and learning about the artwork they contain. It is a cool service, offering very high res images of the paintings, but not providing a machine-friendly access to the data shown. This server provides a wrapper around the HTML page for a painting and exposes part of the information using FOAF and Dublin Core ontologies.

Usage

You can access any painting as follows:

  • http://linkeddata.few.vu.nl/googleart/(path to the painting on googleartproject.com)

Here are two examples (google art page first, RDF second):

  • Night Watch [RDF]
  • The Birth of Venus [RDF]
